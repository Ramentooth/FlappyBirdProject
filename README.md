## Details
### Description
The game would work like regular flappy bird with the user having to jump through holes in bars in order to pass trough the next pole and the user cannot at all touch the poles. As the user progresses through each pillar the score increases by one and the game continues infinitely until the user loses where then the user would be prompted with a game over screen where the highscore is saved and the user can play again.

### Target User
For recreational gamers with extra time on their hands since the controls are pretty self explanitory.

### Planned Features
- different types of birds with different speeds and flap length.
- pipes with two entry points with one scoring the player more points if they can make it in.
- Changing backgrounds as the user progresses.

### Program Structure

- Main file
- Bird class
- Pipe class
- Background class
- Button class

### Challenges

The most challenging part of this project would most likely be the gravity function.
